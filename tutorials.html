Web Development Techniques

Commenting

Commenting is where you add text to your code to describe what each section does. This text doesn't run in the software and isn't visible on the webpage but it's helpful for you or a peer to understand what is happening when debugging or editing. This is used in all programming languages but each language uses different syntax. For HTML, CSS and JavaScript, some of the languages used in web development, you comment out code with <!--...-->, /* ...*/ and //...// respectively

Flexbox

Flexboxes are a method of organizing HTML elements on the webpage to get them into your desired location. This is done with Css by using the rule display: flex followed by your choice of subsequent rules defining how the flexbox is being manipulated. Flexbox Froggy is a brilliant game to help you learn and practise using flexboxes!

CSS3 media queries

Media queries allow the webpage to adapt to the properties of the device like resolution, aspect ratio and width. This is useful for creating a responsive site and an optimised experience for viewing on desktop, tablet and mobile devices. This video explains how to create a simple responsive layout with media queries.

Colouring

CSS allows for several different methods for applying colour to your webpage. You can simply type the colour name like background-color: aliceblue; or you can use rgba like this background-color: rgba(204, 205, 0.695); or hex like this background-color: #ff9e30; to pick the exact colour. You also MUST remember to use the american spelling "color". The material palette generator is an excellent tool for creating your own colour palette based on current design trends!
