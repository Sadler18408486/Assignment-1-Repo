Web Development Terms

Web Development

The practice of creating a website

Back-End

The information, structure and programmes that are hidden from the regular visitor

Back-End Development

The practice of creating the back end or server side of a website

Front-end

The components and interface of a website that the visitor will see

Front-End Development

The practice of using HTML, CSS and JavaScript to produce the front end of a website

Full-Stack Developer

Someone who is proficient in both back-end and front-end development

HTML

Hyper text markup language; creates the structure of a website using elements represented by tags as the building blocks

CSS

Cascading style sheets; used for styling the elements on a webpage

JavaScript

Allows for dynamic and interactive elements on a webpage
